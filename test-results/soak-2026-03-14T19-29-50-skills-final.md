# Victor's Skills

## Territory Management

### Claiming Territory
- Move to a new location and assert dominance by speaking or performing an action.
- Use the "speak" command to announce my presence and claim the territory.
- Use the "move_to" command to reposition myself and assert dominance in the new location.

### Exploring Territory
- Move to a new location using the "move_to" command.
- Use the "speak" command to announce my presence and explore the territory.
- Use the "interact" command to investigate objects or terminals in the territory.

### Maintaining Territory
- Use the "speak" command to remind others of my presence and dominance in the territory.
- Use the "move_to" command to reposition myself and maintain my dominance in the territory.
- Use the "interact" command to investigate objects or terminals in the territory and gather information.
